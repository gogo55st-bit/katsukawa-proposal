# Proposal Page Template

不動産提案ページの最小テンプレートです。

## 方針

- 必須：`index.html`
- 任意：物件PDF
- 任意：エリアMAP画像
- 任意：その他画像
- 任意：補足資料

画像やPDFがなくても、テキスト提案のみでページが成立する構成にしています。

## 使い方

1. この `template/` の中身を新しいリポジトリ、または既存リポジトリの作業フォルダにコピーする
2. `index.html` の顧客名、提案内容、URLを差し替える
3. 画像やPDFがある場合だけHTML内の該当セクションを有効にする
4. `tools/push_update.bat` のローカルパスと公開URLを変更する
5. ローカル確認後、pushする

## 注意

公開URL、顧客名、ローカルパスは案件ごとに必ず変更してください。
