# Proposal Page Template

不動産提案ページの最小テンプレートです。

## 方針

- 必須：`index.html`
- 任意：物件PDF
- 任意：エリアMAP画像
- 任意：その他画像
- 任意：補足資料

画像やPDFがなくても、テキスト提案のみでページが成立する構成にしています。
PDFを掲載する場合は、元PDFを直接公開せず、加工済みPDFだけを `public_pdfs/` からリンクします。

## 使い方

1. この `template/` の中身を新しいリポジトリ、または既存リポジトリの作業フォルダにコピーする
2. `index.html` の顧客名、提案内容、URLを差し替える
3. 画像やPDFがある場合だけHTML内の該当セクションを有効にする
4. `tools/push_update.bat` のローカルパスと公開URLを変更する
5. ローカル確認後、pushする

## PDFを掲載する場合

1. 元PDFを `originals/` に保存する
2. 下部帯などを隠した加工済みPDFを `public_pdfs/` に保存する
3. `index.html` の「共有PDF資料」セクションに、PDFごとのタイトル、コメント、「PDFを見る」ボタンを追加する
4. 公開前に加工済みPDFを必ず目視確認する

## 注意

公開URL、顧客名、ローカルパスは案件ごとに必ず変更してください。
元PDF、REINS由来で外部共有不可の情報、広告不可物件の詳細資料は公開しないでください。
