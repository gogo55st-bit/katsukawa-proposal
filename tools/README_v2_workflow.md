# 勝川さん v2 提案ページ更新手順

## 前提

正式URLは常に以下です。

```text
https://gogo55st-bit.github.io/katsukawa-proposal/
```

現在のv1ページは `archive/v1/` に保存済みです。ルートの `index.html` を更新すると、同じ正式URLの内容が新しい提案に差し替わります。

## 重要方針

- v2の実ページは、必要になってから作成する
- 今回の手順書作成時点では、ルートの `index.html` は再生成しない
- エリアMAPは必須ではない
- 物件PDFも必須ではない
- 画像なし、PDFなし、テキスト提案のみでも進めてよい

## v2作成時の流れ

1. v2で何を提案するか決める
   - 新しい候補物件
   - 追加エリア
   - 条件変更への回答
   - 審査・初期費用・通勤などの補足

2. 必要に応じて素材を用意する
   - エリアMAP画像：必要な場合のみ
   - 物件PDF：必要な場合のみ
   - 物件画像：必要な場合のみ
   - テキストコメント：基本情報として用意

3. `archive/v1/` に現在のv1が保管済みであることを確認する

4. 新しい `index.html` や必要ファイルをリポジトリ直下に配置する
   - 画像やPDFは相対パスで参照する
   - PC内の絶対パスはHTMLに書かない

5. ローカルで表示確認する
   - スマホ幅で読みやすいか
   - PDFなしの場合に不自然な空欄がないか
   - エリアMAPなしの場合にページが成立しているか
   - LINEで送るURLが正式URLになっているか

6. `tools/push_update.bat` を実行する

7. GitHub Pages反映後、正式URLを確認する

```text
https://gogo55st-bit.github.io/katsukawa-proposal/
```

8. LINE送信用文面を作る

## LINE文面の型

```text
勝川さん

追加の候補・補足情報をまとめました。
まずはこちらのページをご確認ください。

https://gogo55st-bit.github.io/katsukawa-proposal/

必要に応じて、気になる物件やエリアを教えてください。
```

## 掲載しないもの

```text
・個人情報
・申込書類
・身分証
・オーナー情報
・管理会社の内部情報
・REINS由来で外部共有不可の情報
・広告不可物件の詳細資料
```
